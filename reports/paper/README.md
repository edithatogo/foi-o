# Paper review reports

Use the role-specific report schema from the validated bundle. Required roles
include Sourceright, red team, devil's advocate, FOI/legal, methods/statistics,
ontology, process/reproducibility, architecture/context, global impact and
Authentext, followed by an adjudicator. The authoring agent cannot be the sole
grader. A score above 995 cannot override a failed critical gate or the human
publication gate.
