# Data rights, ethics, security, and release

Raw archives, derived records, gold/adjudication data, and normative source
packs are separately governed. Public access does not by itself establish
redistribution rights. Derived releases minimise personal information, assess
small-cell and community risk, provide correction/takedown channels, and may
restrict high-risk gold data.

Archived content is untrusted. Parsers are sandboxed and resource-limited;
active content is disabled; raw hashes and transformation lineage are retained;
source text cannot instruct agents or change certification state.

Releases pin archive, code, schema, codebook, source-pack, extractor, and
environment versions and include checksums, provenance, an SBOM where feasible,
and explicit limitations.
