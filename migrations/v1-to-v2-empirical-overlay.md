# V1 to V2 empirical overlay

V2-alpha adds optional governance and assertion records. A V1 record is not
rewritten merely to add a V2 assertion. The new record points to V1 event,
ledger, trace, lineage, legal-source, annotation, or gold identifiers. Automatic
migration may create only neutral linkage stubs; it must not manufacture label
maturity, certification, or evidence dimensions.
