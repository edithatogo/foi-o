# Paper archival policy

arXiv retains public manuscript version history. A GitHub release, Zenodo
version/concept DOI, and OSF project may preserve complementary source packs,
review reports, lockfiles, checksums, SBOMs, and reproducibility capsules.
Hugging Face may host appropriate public derived data, but is not canonical
manuscript source. None of these actions is automated by this repository.

Every future paper ledger entry must record local release ID, public identifier
if any, exact commit, source/data/version locks, citation and reviewer bundle
hashes, generated output hashes, human approver and approval time, changes from
the prior version, and residual limitations. Unknown identifiers remain
`not_verified`; they are never inferred from local files.
